## Overview of teacher controls

CoGuide provides comprehensive controls that put you in charge of your classroom monitoring experience. All controls are designed to be intuitive and non-disruptive to your teaching flow.

## Main control panel

### Start/Stop controls

The primary controls for managing your monitoring session:

<Columns cols={2}>
  <Card title="Start Standby" icon="play">
    **Begin monitoring**
    - Starts real-time participation tracking
    - Activates live overlay display
    - Begins data collection
    - One-click activation
  </Card>
  
  <Card title="Stop Standby" icon="stop">
    **End monitoring**
    - Stops all tracking and data collection
    - Saves session data
    - Closes live overlay
    - Preserves privacy
  </Card>
  
  <Card title="Pause/Resume" icon="pause">
    **Temporary control**
    - Pause tracking without losing data
    - Resume when ready
    - Useful for breaks or transitions
    - Maintains session continuity
  </Card>
  
  <Card title="Emergency stop" icon="exclamation-triangle">
    **Immediate shutdown**
    - Instantly stops all monitoring
    - Clears sensitive data if needed
    - Available at any time
    - Privacy protection
  </Card>
</Columns>

### Overlay controls

Manage the live monitoring display:

#### Position and size
- **Drag to move**: Click and drag overlay to any screen position
- **Resize handles**: Drag corners to adjust overlay size
- **Snap to edges**: Overlay can snap to screen edges for easy positioning
- **Minimize/maximize**: Collapse overlay to small indicator or expand for full view

#### Display options
- **Show/hide metrics**: Toggle individual metrics on/off
- **Transparency slider**: Adjust overlay opacity (10-90%)
- **Color themes**: Choose from multiple color schemes
- **Font size**: Adjust text size for better visibility

#### Layout customization
- **Metric arrangement**: Drag metrics to reorder display
- **Compact mode**: Condensed view for minimal screen space
- **Full mode**: Detailed view with all available metrics
- **Custom layout**: Create your own metric arrangement

## Settings and preferences

### Privacy settings

<Steps>
  <Step title="Data location">
    Choose where your data is stored:
    - **Local only**: Data stays on your device
    - **School cloud**: Data stored in your school's cloud
    - **Hybrid**: Local processing with cloud backup
  </Step>
  
  <Step title="Retention policy">
    Set how long to keep classroom data:
    - **7 days**: Short-term retention
    - **30 days**: Standard retention (recommended)
    - **90 days**: Extended retention
    - **Custom**: Set your own retention period
  </Step>
  
  <Step title="Offline mode">
    Configure offline operation:
    - **Enable offline**: Work without internet connection
    - **Sync when connected**: Upload data when internet available
    - **Local processing only**: Never use cloud services
  </Step>
</Steps>

### Monitoring preferences

#### Participation tracking
- **Hand raise detection**: Enable/disable automatic hand raise counting
- **Talk time tracking**: Turn on/off teacher and student talk time monitoring
- **Turn counting**: Enable/disable participation turn tracking
- **Wait time alerts**: Set up notifications for extended wait times

#### Alert configuration
- **Visual alerts**: Choose alert types and colors
- **Sound notifications**: Enable/disable audio alerts
- **Alert thresholds**: Set custom thresholds for different metrics
- **Alert frequency**: Control how often alerts appear

#### Display preferences
- **Metric selection**: Choose which metrics to display
- **Update frequency**: Control how often metrics refresh
- **Animation effects**: Enable/disable visual transitions
- **Accessibility options**: High contrast, large text, screen reader support

### Advanced settings

#### Detection sensitivity
- **Hand raise sensitivity**: Adjust detection threshold for hand raises
- **Voice detection**: Fine-tune microphone sensitivity
- **Movement detection**: Control sensitivity for participation gestures
- **Calibration tools**: Access advanced calibration options

#### Data export
- **Export formats**: Choose from CSV, PDF, or JSON formats
- **Export frequency**: Set up automatic data exports
- **Data filtering**: Select which data to include in exports
- **Privacy controls**: Ensure exports comply with privacy policies

## Quick access controls

### Keyboard shortcuts

Speed up common tasks with keyboard shortcuts:

<AccordionGroup>
  <Accordion title="Basic controls">
    - **Space**: Start/stop monitoring
    - **P**: Pause/resume monitoring
    - **Escape**: Emergency stop
    - **Tab**: Cycle through overlay elements
  </Accordion>
  
  <Accordion title="Display controls">
    - **H**: Hide/show overlay
    - **M**: Minimize/maximize overlay
    - **T**: Toggle transparency
    - **R**: Reset overlay position
  </Accordion>
  
  <Accordion title="Settings access">
    - **Ctrl+S**: Open settings
    - **Ctrl+P**: Privacy settings
    - **Ctrl+A**: Alert settings
    - **Ctrl+D**: Display preferences
  </Accordion>
</AccordionGroup>

### Touch controls (for tablets)

If using CoGuide on a tablet, take advantage of touch controls:

- **Tap overlay**: Access quick settings
- **Pinch to zoom**: Adjust overlay size
- **Swipe gestures**: Navigate between different views
- **Long press**: Access advanced options

## Session management

### Starting a new session

<Steps>
  <Step title="Pre-session check">
    - Verify camera and microphone are working
    - Check seat map calibration (if using)
    - Review privacy settings
    - Position overlay appropriately
  </Step>
  
  <Step title="Start monitoring">
    - Click "Start Standby" or press Space
    - Verify overlay appears and metrics are updating
    - Check that all desired metrics are visible
    - Confirm alerts are working properly
  </Step>
  
  <Step title="Monitor during class">
    - Check metrics periodically during class
    - Adjust settings if needed
    - Use pause/resume as appropriate
    - Address any technical issues quickly
  </Step>
</Steps>

### Ending a session

<Steps>
  <Step title="Stop monitoring">
    - Click "Stop Standby" or press Space
    - Confirm data is saved
    - Review session summary if available
    - Close overlay if desired
  </Step>
  
  <Step title="Data handling">
    - Data is automatically saved according to your settings
    - Review any alerts or notifications
    - Export data if needed
    - Clear sensitive data if required
  </Step>
  
  <Step title="Post-session review">
    - Access analytics and insights
    - Review participation patterns
    - Plan adjustments for next class
    - Share insights with students if appropriate
  </Step>
</Steps>

## Troubleshooting controls

### Common control issues

<AccordionGroup>
  <Accordion title="Overlay not responding">
    **Symptoms:**
    - Overlay won't move or resize
    - Controls not working
    - Display frozen or unresponsive
    
    **Solutions:**
    - Restart CoGuide application
    - Check for software updates
    - Verify display settings
    - Contact support if persistent
  </Accordion>
  
  <Accordion title="Settings not saving">
    **Symptoms:**
    - Changes don't persist between sessions
    - Settings revert to defaults
    - Preferences not applied
    
    **Solutions:**
    - Check file permissions
    - Ensure sufficient storage space
    - Restart application after changes
    - Check for software conflicts
  </Accordion>
  
  <Accordion title="Controls not accessible">
    **Symptoms:**
    - Can't access settings menu
    - Keyboard shortcuts not working
    - Touch controls unresponsive
    
    **Solutions:**
    - Check device compatibility
    - Update to latest version
    - Restart device if needed
    - Contact support for assistance
  </Accordion>
</AccordionGroup>

### Emergency procedures

#### Immediate shutdown
If you need to immediately stop all monitoring:

1. **Press Escape** or click Emergency Stop
2. **Confirm shutdown** when prompted
3. **Verify data is cleared** if privacy is a concern
4. **Restart application** when ready to resume

#### Data protection
If you need to protect sensitive data:

1. **Stop monitoring** immediately
2. **Access privacy settings**
3. **Clear sensitive data** using built-in tools
4. **Verify data is removed** from device and cloud
